Time Series¶

There is no real Time Series Algorithm. They are Regressions and Classifications. For example, predicting the number of Forest fires in Brazil is a Regression model which will be considered as a Time Series model.

A Time Serie is a dataset which has a timestamp label. Let's consider the example of the Amazon dataset.

from verticapy import *

The feature 'date' indicates us the temporality of the dataset. To do predictions on Time Series, we most of the time use previous values (called lags). To identify seasonality of a process, we use Autocorrelation Plots.

In [7]:
amazon.autocorr(ts = "date", 
                column = "number", 
                by = ["state"], 
                p = 48)

We can see the seasonality of the process and we can easily use the past values to predict the future.

In [3]:
from verticapy.learn.linear_model import LinearRegression

model = LinearRegression("LR_amazon")
model.drop()
model.fit("amazon_clean", ["lag_number_{}".format(i) for i in range(1,25)], "number")
