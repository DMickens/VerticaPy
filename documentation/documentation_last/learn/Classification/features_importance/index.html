classifier.features_importance¶

In [ ]:
classifier.features_importance()

Computes the model features importance.

Returns¶

tablesample : An object containing the result. For more information, check out utilities.tablesample.

Example¶

In [6]:
from verticapy.learn.ensemble import RandomForestClassifier
model = RandomForestClassifier(name = "public.RF_iris",
                               n_estimators = 20,
                               max_features = "auto",
                               max_leaf_nodes = 32, 
                               sample = 0.7,
                               max_depth = 3,
                               min_samples_leaf = 5,
                               min_info_gain = 0.0,
                               nbins = 32)
model.fit("public.iris", 
          ["PetalLengthCm", "PetalWidthCm", "SepalWidthCm", "SepalLengthCm"], 
          "Species")
model.features_importance()
