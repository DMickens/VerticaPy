Unsupervised¶

This example will show you how to use the different methods of an Unsupervised Model. We will use the Iris dataset.

iris = load_iris()
print(iris)

Let's create a KMeans model to segment the data into clusters.

In [20]:
from verticapy.learn.cluster import KMeans
model = KMeans("public.KMeans_iris", n_cluster = 3)
model.fit("public.iris", ["PetalWidthCm", "PetalLengthCm"])
