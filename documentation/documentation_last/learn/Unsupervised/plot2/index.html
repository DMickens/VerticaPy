unsupervised.plot¶

In [ ]:
unsupervised.plot(X: list = [], 
                  tablesample: float = -1)

Draws the model if the number of predictors is 2 or 3.

Parameters¶

X
list
✓
 List of the predictors to display. The score will stay the same.
tablesample
float
✓
 Sample of data to display. If this number is not between 0 and 1 all the data points will be displayed.

Example¶

In [4]:
from verticapy.learn.neighbors import LocalOutlierFactor
model = LocalOutlierFactor(name = "public.LOF_iris")
model.fit("public.iris", ["PetalLengthCm", "SepalLengthCm"])
model.plot()
