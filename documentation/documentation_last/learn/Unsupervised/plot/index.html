unsupervised.plot¶

In [ ]:
unsupervised.plot(voronoi: bool = False)

Draws the unsupervised model.

Parameters¶

Name Type Optional Description
input_relation
str
✓
 If set to True, a voronoi plot will be drawn. It is only available for KMeans using 2 predictors.

Example¶

In [3]:
from verticapy.learn.cluster import KMeans
model = KMeans(name = "public.KMeans_iris",
               n_cluster = 5)
model.fit("public.iris", ["PetalLengthCm", "SepalLengthCm"])
# Regular Plot
model.plot()
