load_smart_meters¶

In [ ]:
load_smart_meters(cursor = None, 
                  schema: str = 'public',
                  name: str = 'smart_meters')

Ingests the smart meters dataset in the Vertica DB (Dataset ideal for TS and Regression). If a table with the same name and schema already exists, this function will create a vDataFrame from the input relation.

Parameters¶

Name Type Optional Description
cursor
DBcursor
✓
 Vertica DB cursor.
schema
str
✓
 Schema of the new relation. The default schema is public.
name
str
✓
 Name of the new relation.

Returns¶

vDataFrame : the smart meters vDataFrame.
