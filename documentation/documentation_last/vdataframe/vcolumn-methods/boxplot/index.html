vDataFrame[].boxplot¶

In [ ]:
vDataFrame[].boxplot(by: str = "", 
                     h: float = 0, 
                     max_cardinality: int = 8, 
                     cat_priority: list = [])

Draws the vcolumn Box Plot.

Parameters¶

Name Type Optional Description
by
str
✓
 vcolumn used to partition the data.
h
float
✓
 Interval width if the vcolumn is numerical or of type date like. Optimized h will be computed if the parameter is empty or invalid.
max_cardinality
int
✓
 Maximum number of vcolumn distinct elements to be used as categorical. The less frequent elements will be gathered together to create a new category : 'Others'.
cat_priority
list
✓
 List of the different categories to consider when drawing the Box Plot. The other categories will be filtered.

Returns¶

vDataFrame : self.parent
