vDataFrame.autocorr¶

In [ ]:
vDataFrame.autocorr(ts: str, 
                    column: str, 
                    by: list = [],
                    p = 12,
                    method: str = "pearson", 
                    cmap: str = "",
                    round_nb: int = 3, 
                    focus: bool = False,
                    show: bool = True,
                    matrix: bool = False)

Computes the correlations of the input vcolumn and its lags.

Parameters¶

Name Type Optional Description
ts
str
❌
 TS (Time Series) vcolumn used to order the data. It can be of type date or a numerical vcolumn.
column
str
❌
 Input vcolumn used to compute the Auto Correlation Plot.
by
list
✓
 vcolumns used in the partition.
p
int / list
✓
 Int equals to the maximum number of lag to consider during the computation or List of the different lags to include during the computation. p must be positive or a list of positive integers.
unit
str
✓
 Unit used to compute the lags.
  • rows : Natural lags.
  • else : Any time unit, for example you can write 'hour' to compute the hours lags or 'day' to compute the days lags.
method
str
✓
 Method used to compute the correlation.
  • pearson : Pearson correlation coefficient (linear).
  • spearmann : Spearmann correlation coefficient (monotonic - rank based).
  • kendall : Kendall correlation coefficient (similar trends).
  • cramer : Cramer's V (correlation between categories).
  • biserial : Biserial Point (correlation between binaries and a numericals).
cmap
str
✓
 Color Map.
round_nb
int
✓
 Round the coefficient using the input number of digits. It is used only if parameter 'matrix' is set to True.
focus
bool
✓
 Computes a correlation vector instead of the matrix. It is used only if parameter 'matrix' is set to True.
show
bool
✓
 If set to True, the Auto Correlation Plot will be drawn using Matplotlib.
matrix
bool
✓
 If set to True, draw a correlation matrix instead of a regular plot.

Returns¶

tablesample : An object containing the result. For more information, check out utilities.tablesample.

# Autocorrelation Plot for each 'year' lag using Spearman coefficients
# p = 48: it will compute 48 'months' lags
amazon.autocorr(ts = "date", 
                column = "number", 
                p = 48,
                by = ["state"],
                unit = "month",
                method = "spearman")
