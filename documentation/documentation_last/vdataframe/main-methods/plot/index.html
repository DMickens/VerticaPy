vDataFrame.plot¶

In [ ]:
vDataFrame.plot(ts: str,
                columns: list = [], 
                start_date: str = "",
                end_date: str = "")

Draws the Time Series.

Parameters¶

Name Type Optional Description
ts
str
❌
 TS (Time Series) vcolumn used to order the data. The vcolumn type must be date like (date, datetime, timestamp...) or numerical.
columns
list
✓
 List of the vcolumns names. If empty, all the numerical vcolumns will be used.
start_date
str
✓
 Input Start Date. For example, time = '03-11-1993' will filter the data when 'ts' is lesser than November 1993 the 3rd.
end_date
str
✓
 Input End Date. For example, time = '03-11-1993' will filter the data when 'ts' is greater than November 1993 the 3rd.

Returns¶

vDataFrame : self
