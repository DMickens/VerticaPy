vDataFrame.scatter¶

In [ ]:
vDataFrame.scatter(columns: list,
                   catcol: str = "",
                   max_cardinality: int = 3,
                   cat_priority: list = [],
                   with_others: bool = True,
                   max_nb_points: int = 20000,
                   bbox: list = [],
                   img: str = "")

Draws the Scatter Plot of the input vcolumns.

Parameters¶

Name Type Optional Description
columns
list
❌
 List of the vcolumns names. The list must have two or three elements.
catcol
str
✓
 Categorical vcolumn used to label the data.
max_cardinality
int
✓
 Maximum number of distinct elements for 'catcol' to be used as categorical. The less frequent elements will be gathered together to create a new category: 'Others'.
cat_priority
list
✓
 List of the different categories to consider when labeling the data using the vcolumn 'catcol'. The other categories will be filtered.
with_others
bool
✓
 If set to false and the cardinality of the vcolumn 'catcol' is too big then the less frequent element will not be merged to another category and they will not be drawn.
max_nb_points
int
✓
 Maximum number of points to display.
bbox
list
✓
 List of 4 elements to delimit the boundaries of the final Plot. It must be similar the following list: [xmin, xmax, ymin, ymax]
img
str
✓
 Path to the image to display as background.

Returns¶

vDataFrame : self

# img parameter can be used to display a background image
from verticapy import *
# Africa Dataset
africa = vDataFrame("africa_education")
africa = africa.groupby(["country_long", "lat", "lon"])
# displaying schools in Africa
africa.scatter(["lon", "lat"],
               catcol = "country_long",
               max_cardinality = 100,
               img = "img/africa.png")
