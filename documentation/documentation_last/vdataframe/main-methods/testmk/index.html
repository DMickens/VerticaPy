vDataFrame.testmk¶

In [ ]:
vDataFrame.testmk(ts: str, 
                  column: str, 
                  k = 1000,
                  alpha: float = 0.05,
                  print_info: bool = True)

Mann Kendall test (Time Series trend).

Name Type Optional Description
ts
str
❌
 vcolumn used as timeline. It will be used to order the data. It can be a numerical or type date like (date, datetime, timestamp...) vcolumn.
column
str
❌
 Input vcolumn to test.
k
int/str
✓
 Number of elements to take into account to see the trend. It can be also a time interval. For example: if k = 1000, only the last 1000 elements will be taken into account whereas if k = '1 day', the test will identify the last day trend compared to the past.
alpha
float
✓
 Significance Level. Probability to accept H0.
print_info
bool
✓
 If set to True, displays all the test information.

Returns¶

tuple :
[0] result of the test : True if H0 was rejected,
[1] ZMK : Mann Kendall ZMK Index,
[2] p_value : p value of the coefficient
