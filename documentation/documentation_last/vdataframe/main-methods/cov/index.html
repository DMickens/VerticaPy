vDataFrame.cov¶

In [ ]:
vDataFrame.cov(columns: list = [], 
               cmap: str = "",
               focus: str = "",
               show: bool = True)

Computes the Covariance Matrix of the vDataFrame.

Parameters¶

Name Type Optional Description
columns
list
✓
 List of the vcolumns names. If empty, all the numerical vcolumns will be used.
cmap
str
✓
 Color Map.
focus
str
✓
 Focus the computation on only one vcolumn.
show
bool
✓
 If set to True, the Covariance Matrix will be drawn using Matplotlib..

Returns¶

tablesample : An object containing the result. For more information, check out utilities.tablesample.
