vDataFrame.boxplot¶

In [ ]:
vDataFrame.boxplot(columns: list = [])

Draws the Box Plot of the input vcolumns.

Parameters¶

Name Type Optional Description
columns
list
✓
 List of the vcolumns names. If empty, all the numerical vcolumns will be used.
