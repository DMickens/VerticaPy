available_auto_connection¶

In [ ]:
available_auto_connection()

Displays all the available auto connections.

Returns¶

list : all the available auto connections.

Example¶

In [2]:
from verticapy.connections.connect import available_auto_connection
available_auto_connection()

The available connections are the following: VerticaDSN, VML, VML2
Out[2]:
['VerticaDSN', 'VML', 'VML2']
